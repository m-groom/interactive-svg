Interactive Climate Visualization App

Welcome to my interactive climate visualization application. This tool provides advanced visualizations of climate predictability patterns using two complementary approaches:

Available Visualizations

  • Markov Chain Visualizations Interactive transition graphs showing probabilistic state transitions across 1-24 month lead times. Features hover tooltips, click modals with detailed statistics, and cluster video playback.
  • Directed Acyclic Graph (DAG) Visualizations Multi-level hierarchical transition graphs displaying climate predictability patterns across lead times. Includes level-based spatial mapping and climatological probability calculations.
  • Case Studies Detailed case study visualizations and analyses (in development).

Technical Features

The application is built with a modular ES6 JavaScript architecture featuring:

  • 15 JavaScript modules with inheritance-based design for code reuse
  • 96 SVG files across both visualization types
  • 450+ MP4 cluster videos with variable availability per lead time
  • 598 historical climate images from 2000-2024 (original and detrended)
  • Mobile-responsive design with touch optimization
  • Dark mode support with automatic system preference detection

For more information about the development of this application, see the blog post on the main website.
